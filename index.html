HTML, CSS y JavaScript:

Son los tres pilares fundamentales del desarrollo web que trabajan juntos para crear páginas web dinámicas e interactivas.
HTML se encarga de la estructura y el contenido, CSS del diseño y la presentación, y JavaScript de la interactividad y funcionalidad.

Definiciones:

HTML (HyperText Markup Language)

Es el lenguaje básico utilizado para crear y estructurar contenido en la web. Consiste en una serie de etiquetas que definen cómo se deben mostrar los elementos en una página. Por ejemplo, h1 se usa para encabezados, p para párrafos, a para enlaces y img para imágenes. HTML organiza el contenido de manera que los navegadores puedan mostrarlo correctamente, creando la estructura fundamental de cualquier página web.

CSS (Cascading Style Sheets)

Es un lenguaje utilizado para describir la apariencia y el diseño de una página web escrita en HTML. Mientras que HTML define la estructura del contenido, CSS se encarga de darle estilo, como colores, fuentes, márgenes y disposición de los elementos. Utiliza reglas de estilo que se aplican a los elementos HTML, permitiendo cambiar su apariencia de forma flexible y consistente. Por ejemplo, con CSS se puede cambiar el color de fondo, el tamaño de la fuente o la alineación de un texto.

JavaScript (Program Language)

Es un lenguaje de programación utilizado para hacer que las páginas web sean interactivas. A diferencia de HTML y CSS, que se enfocan en la estructura y el estilo, JavaScript permite agregar funcionalidades dinámicas, como responder a clics, validar formularios, mostrar mensajes emergentes o actualizar contenido sin recargar la página. Se ejecuta en el navegador y puede interactuar con los elementos HTML y CSS para modificar el comportamiento de la página en tiempo real.

Analogía con una casa:

HTML

Estructura de la casa.

Es como los cimientos, las paredes, techos, y distribución de las habitaciones.

HTML define la estructura básica de una página web: qué elementos hay (título, párrafos, botones, imágenes, etc.), pero sin preocuparse por cómo se ven.

CSS

Decoración y estilo de la casa.

Es la pintura de las paredes, el tipo de piso, los muebles, las cortinas, el diseño de la cocina.

CSS se encarga de cómo se ve la página: colores, fuentes, márgenes, alineaciones, estilos responsivos… lo visual.

JavaScript

Los sistemas eléctricos y electrónicos de la casa.

Es como los interruptores que encienden luces, las puertas automáticas, el termostato inteligente.

JavaScript añade funcionalidad e interactividad haciendo que pasen cosas como respuesta a un evento.
